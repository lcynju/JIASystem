This repository is for the following research paper:

Yuan Zhang, Chuanyi Li, Yu Sheng, Jidong Ge, Bin Luo: Judicial intelligent assistant system: Extracting events from Chinese divorce cases to detect disputes for the judge. Expert Syst. J. Knowl. Eng. 41(7) (2024)

It includes the dataset for constructing the models, and the source code of training and the system.
